実行部
___________________________________
    ROOT_DIRにいるものは, システムの本体(実行部)である.

    実行部は, あらゆるモジュール, サービス, クライアントサイドのスタイル, ライブラリを駆使して, 動作する.

    ここにあるものは, ユーザと直接の関係を持つ.

    * ただ, このルートでないと正常に動作できない(パスが通らないなど)場合はこの限りでない.
